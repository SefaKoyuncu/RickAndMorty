# RickAndMorty

### Project Information

- The first activity lists rick and morty characters. You can also find characters by searching.

- In the second activity, more detailed information about the characters can be accessed.

### Techs.

-Java
 
-Retrofit

-Picasso

-DataBinding  
 
-Recyclerview-Cardview

-Lottie

-Videoview

-Clean Code
